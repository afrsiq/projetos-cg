Desenho de Linhas e Triângulos com Algoritmo de Bresenham

Modo Atual: LINHAS (R) - Clique para definir pontos da linha

Instruções:

  • Modo Linhas (R/r): 2 cliques para desenhar uma linha
  • Modo Triângulos (T/t): 3 cliques para desenhar um triângulo
  • Pressione teclas de 0 a 9 para mudar a cor
  • Pressione C para limpar a tela

Cores: 0-Preto, 1-Azul, 2-Vermelho, 3-Verde, 4-Amarelo, 5-Magenta, 6-Ciano, 7-Laranja, 8-Roxo, 9-Marrom
